For C++ coding:
g++ compiler
$: rm outMainCompiled; g++ -std=c++14 -Werror -o outMainCompiled -c main.cpp; ls -la;
-Werror     (Make all warnings into errors)
-std=c++14  (use of 2014 C++ spec  )
-o out      (output file named out)
-c          (to not do the linking)
g++ linker
$: g++ -std=c++14 -Werror -o outLinked -g outCompiled
-g          (do the linking) ?
gdb debuger
